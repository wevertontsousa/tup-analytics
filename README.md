# Análise de Telefones de Uso Público

Sistema web interna para consolidar dados operacionais e gerar dashboards e relatórios sobre a operação de Telefones de Uso Público (TUPs).

## Objetivo

Substituir controles manuais em Excel e e-mails por um sistema web simples que permita visualizar indicadores, acompanhar a execução dos serviços e gerar informações para análise operacional.
